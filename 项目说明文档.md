# 基金TA每日简报系统

## 项目概述

本系统是一款面向公募基金TA岗（注册登记）从业人员的每日资讯汇总工具，通过自动化爬取、过滤、生成和推送，为用户每日提供监管动态、市场行情和财经新闻的一站式简报服务。

**核心功能：**
- 爬取证监会、中国结算、基金业协会等监管机构官网
- 爬取新华网、人民网等权威财经媒体
- 通过关键词过滤筛选与基金TA业务相关的资讯
- 使用DeepSeek API生成行情简报和资讯摘要
- 生成HTML格式简报并支持邮件推送
- 候选池机制保留历史优质资讯，每日重新筛选

## 模块功能说明

### 1. 爬虫模块（src/crawlers/）

#### 1.1 基础爬虫（base_crawler.py）
- **功能**：提供requests和playwright两种爬取引擎，支持自动降级
- **特性**：
  - 请求间隔控制（默认3秒）
  - 超时重试机制（默认3次）
  - User-Agent伪装
  - Headless模式支持

#### 1.2 监管资讯爬虫（regulator_crawler.py）
- **数据来源**：
  - 证监会（http://www.csrc.gov.cn）
  - 中国结算-要闻动态（http://www.chinaclear.cn）
  - 中国结算-通知公告
  - 中国结算-业务通知
  - 基金业协会（https://www.amac.org.cn）
- **解析策略**：BeautifulSoup CSS选择器
- **日期验证**：14天内有效
- **内容获取**：逐条抓取详情页正文（最多3000字）
- **去重机制**：基于URL去重

#### 1.3 行情数据爬虫（market_crawler.py）
- **数据来源**：akshare库（支持上证、深证、沪深300、创业板、中证全债、国债指数）
- **采集指标**：开盘、收盘、最高、最低、成交量、涨跌幅
- **数据存储**：原始数据+统计结果

#### 1.4 新闻快讯爬虫（news_crawler.py）
- **数据来源**：
  - 新华网财经（http://www.xinhuanet.com/fortune/）
  - 人民网财经（http://finance.people.com.cn/）
- **解析策略**：各站点专用解析器

### 2. 数据过滤模块（src/filters/data_filter.py）

#### 2.1 监管资讯过滤规则
- **包含关键词**（与基金TA业务相关）：
  - TA类：TA、注册登记、份额、清算、交收、代销、销售机构、基金账户、确权、分红、折算、合并、分拆、申购、赎回、转换、定投、认购、募集
  - 产品类：资产管理、REITs、ETF、LOF、货币基金、债券基金

- **排除关键词**：
  - 股票、期货、期权、证券交易、上市公司、IPO、配股、增发、可转债、科创板、创业板注册制
  - 学习贯彻、党的、党建、纪检、八项规定、民主生活会、调研、问答、致辞、会见、论坛、投资者保护

- **过滤流程**：
  1. 标题长度≥8字
  2. 正文长度≥100字
  3. 不含排除关键词
  4. 含至少一个包含关键词
  5. 按关键词匹配次数打分并降序排列

#### 2.2 新闻快讯过滤规则
- **包含关键词**：央行、货币政策、证监会、基金、A股、上市公司、经济政策、财政政策、降息、降准、汇率、国债、人工智能、新能源、半导体、资本市场
- **排除关键词**：娱乐、明星、体育、足球、篮球、民生、育儿、美食、旅游
- **过滤流程**：同时满足包含关键词且不含排除关键词

### 3. 简报生成模块（src/briefing/briefing_generator.py）

- **输出格式**：HTML单文件（可浏览器直接打开）
- **页面结构**：
  - 今日行情（表格+AI总结）
  - 资讯速递（Tab切换：监管资讯/新闻快讯）
- **展示规则**：
  - 每类资讯最多展示2篇
  - 按filter_score降序排列
  - 标注"按基金TA业务关联度排序，每日精选展示"
- **样式特性**：
  - 紫色渐变主题
  - 响应式布局
  - 文章悬停动效
  - Tab切换高亮

### 4. 工具模块（src/utils/）

#### 4.1 日志模块（logger.py）
- 按天轮转，保留30天
- 同时输出到控制台和文件

#### 4.2 配置加载器（config_loader.py）
- 从config.yaml加载非敏感配置
- 从.env加载敏感配置（API密钥、邮箱密码）

#### 4.3 数据持久化（data_saver.py）
- 保存原始数据（按日期分目录）
- 保存处理后数据
- 保存简报HTML
- 候选池管理（加载/保存/清理）

#### 4.4 DeepSeek客户端（deepseek_client.py）
- 行情总结生成（50字以内）
- 资讯摘要生成（默认50字）
- API失败时本地降级方案

## 运行步骤

### 方式一：运行完整流程

```bash
python run_full_pipeline.py
```

**流程说明**：
1. 采集监管资讯（证监会、中国结算、基金业协会）
2. 采集行情数据（akshare）
3. 采集新闻快讯（新华网、人民网）
4. 数据过滤（关键词过滤+评分）
5. 候选池合并与精选
6. 生成简报HTML

### 方式二：单独运行爬虫

```bash
python run_all_crawlers.py
```

### 方式三：定时任务（需自行配置Windows计划任务或Linux cron）

```bash
# 建议设置每日早上8点运行
python run_full_pipeline.py
```

## 候选池机制

### 工作原理

1. **每日合并**：新爬取的资讯与候选池已有数据按URL去重合并
2. **清理过期**：自动清理超过14天的历史数据
3. **精选排序**：按filter_score降序排列，取前2名进入简报
4. **留存机制**：未选中资讯留在候选池，次日仍有机会被选中

### 配置参数

```yaml
data:
  pool:
    max_display: 2        # 简报每类展示数量
    retention_days: 14    # 候选池保留天数
```

## 注意事项

1. **合规要求**：
   - 遵守网站robots.txt协议
   - 请求间隔不小于3秒
   - 仅爬取公开数据
   - 不处理cookie或登录态

2. **API限制**：
   - DeepSeek API有调用频率限制
   - 建议配置API Key以获得更好的摘要质量

3. **数据时效**：
   - 监管资讯仅保留14天内数据
   - 候选池机制确保优质历史资讯不被遗漏

4. **常见问题**：
   - Playwright无法启动：运行`playwright install chromium`
   - akshare数据获取失败：检查网络或数据源是否可用
   - 邮件发送失败：检查SMTP配置和邮箱授权码
